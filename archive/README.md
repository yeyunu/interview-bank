# Agent 面试历史资料索引

这个目录保存 Agent / RAG / MCP 相关的历史面试资料。不同文件代表不同加工层级，建议按“完整去重版 → 回答稿 → 宽松版 → OCR 全文”的顺序使用。

## 文件说明

### `Agent面试题_完整去重版.md`

- 从 16 份候选面经、63 张图片 OCR 和 233 条候选片段中整理出 179 道主问题。
- 已清除广告、日期、标题和答案噪声，修复明显 OCR 错误，并把同一主题的连续追问合并到主问题下。
- 使用 `## 分类编号：问题` 格式，可直接供 GPT 按题抽取。
- 这是历史题库的默认复习和抽题入口。

### `Agent面试题_按方向整理.md`

- 67 道清洗后的题目，来自 16 位候选人的 63 张面试记录图片。
- 分为 RAG/检索、Agent/框架、Agent 工具调用、评估/观测和大模型应用五个方向。
- 适合作为日常复习入口；为了少噪声做过清洗，因此可能存在漏题。

### `腾讯PCG候选1_套MindClaw回答.md`

- 从候选 1 的问题中整理出 8 道项目型追问。
- 回答结合 MindClaw / ClawCodex 的真实实现，主动区分“当前已经实现”和“未来可以设计”，避免把不存在的能力说成项目现状。
- 适合练习项目介绍、RAG、检索、复杂文档、多 Agent、回退与通信机制等连续追问。

### `Agent面试题_宽松版_少漏题.md`

- 233 条疑似题目或答案片段。
- 目标是尽量不漏内容，因此保留了一部分 OCR 噪声、重复项和非问题片段。
- 适合从整理版复习完以后查漏补缺，不建议从头背诵。

### `Agent面试题_OCR全文.md`

- 16 位候选人的 OCR 全文。
- 未做题目级清洗，保留最多原始上下文。
- 适合检查题目原意、定位漏题或修正 OCR 识别错误。

## 推荐使用方式

1. 从完整去重版选择一个主题复习或让 GPT 随机抽题。
2. 把需要长期训练的题目整理到 `../questions/YYYY-MM-DD.md`。
3. 用腾讯 PCG 回答稿练习把通用题落到自己的项目事实与工程取舍上。
4. 遇到题意不完整时，依次查询按方向整理版、宽松版和 OCR 全文。
