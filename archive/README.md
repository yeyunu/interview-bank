# Agent 面试历史资料索引

这个目录保存 Agent / RAG / MCP 相关的历史面试整理成果。日常复习使用完整去重版，项目表达训练使用腾讯 PCG 回答稿。

## 文件说明

### `Agent面试题_完整去重版.md`

- 从 16 份候选面经、63 张图片 OCR 和 233 条候选片段中整理出 179 道主问题。
- 已清除广告、日期、标题和答案噪声，修复明显 OCR 错误，并把同一主题的连续追问合并到主问题下。
- 使用 `## 分类编号：问题` 格式，可直接供 GPT 按题抽取。
- 这是历史题库的默认复习和抽题入口。

### `腾讯PCG候选1_套MindClaw回答.md`

- 从候选 1 的问题中整理出 8 道项目型追问。
- 回答结合 MindClaw / ClawCodex 的真实实现，主动区分“当前已经实现”和“未来可以设计”，避免把不存在的能力说成项目现状。
- 适合练习项目介绍、RAG、检索、复杂文档、多 Agent、回退与通信机制等连续追问。

## 推荐使用方式

1. 从完整去重版选择一个主题复习或让 GPT 随机抽题。
2. 把需要长期训练的题目整理到 `../questions/YYYY-MM-DD.md`。
3. 用腾讯 PCG 回答稿练习把通用题落到自己的项目事实与工程取舍上。
4. 需要核对原始措辞时，查询本地源目录或 Git 历史。
